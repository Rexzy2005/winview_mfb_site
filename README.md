# WinView Microfinance Bank

A high-end, scroll-driven, digital-first banking experience built for the Nigeria Police Cooperative and every Nigerian.

## 🚀 Experience Highlights

- **Cinematic Visuals**: Dark Green & Lemon Green palette with glassmorphism and neon glow accents.
- **Scroll-Driven Storytelling**: Text reveals, staggered animations, and timeline drawing as you scroll.
- **Micro-interactions**: Custom floating cursor with hover states and reactive timezone widgets.
- **Horizontal Scroll**: Immersive "Lifestyle" section exploring user stories in a horizontal plane.
- **Security-First**: Integrated trust badges and security cues (NDIC, CBN, Biometric).

## 🛠️ Tech Stack

- **Next.js 15+** (App Router & React Server Components)
- **Tailwind CSS v3** (Utility-first styling, Flex/Grid Layouts, Gradients & Blur filters)
- **GSAP (GreenSock)** (High-performance ScrollTrigger & Timeline Animations)
- **Framer Motion** (Component-level micro-interactions & reactive states)
- **Iconify** (Robust, minimal SVG icon packs including Solar and Lucide)
- **React Hooks** (Performant state-driven interactivity like IntersectionObservers)

## 🎨 Design System

- **Background**: Morphing gradients with floating floating particles.
- **Typography**: Inter (Body) & Outfit (Headings).
- **Glassmorphism**: 12px blur panels with subtle borders.

## 📱 Mobile-First

Designed to be lightweight and responsive, ensuring smooth animations across all devices.
